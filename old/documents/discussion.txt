1.  **The "Live Fragment" Layer (B-Style):** Active agents write to ephemeral JSON "Delta" files. This allows for the "Parallel PR" workflow and easy manual review (JTBD 3).

3.  **The "Context Scoper":** When an agent wakes up, it reads from the last *Checkpoint* plus any *Live Fragments* relevant to its task.

This solves:
- **Write Contention:** No direct DB writes; only delta appends.
- **Forking:** You only need to clone the last Checkpoint and start new Deltas.
- **Performance:** No need to rehydrate the entire history; just the last Checkpoint.
