**Instruction:**
Before writing any code for a UI, if you have picked or been asked to use a specific design system, you MUST first use the `read_file` tool to read its corresponding specification file located in the `design_systems` directory. Once you read the file, strictly incorporate its design system philosophy, tokens, and component stylings described in the prompt to create a beautiful, cohesive, and fully realized aesthetic.
