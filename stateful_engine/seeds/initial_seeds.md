# Seeds: Initializing the Engine

These "seeds" show how the fragmented concepts from the old `documents` folder are transformed into configurations for the master engine.

## Seed 1: The Bio-Archive (Fiction/Worldbuilding)
**Description:** A procedural encyclopedia of an organic-tech civilization.
- **Root Node:** "The Great Spore" (ID: SEED-001)
- **Constraint:** "All technology must be breathable."
- **Initial Agents:** Weaver (Tone: Biological/Whimsical), Archivist (Focus: Genetic Lore).

## Seed 2: Repo-Revive (Utility/Self-Improving)
**Description:** A system that modernizes legacy Java code.
- **Root Node:** "Project Root" (Path: `/app/src`)
- **Constraint:** "Zero regressions allowed. 100% test coverage for mutated lines."
- **Initial Agents:** Architect (Logic: Dependency Mapping), Weaver (Focus: Modern Java Patterns).

## Seed 3: Anomaly Logbook (Research/Deep-Niche)
**Description:** A recursive investigation into "The Silence" in deep space.
- **Root Node:** "Station Alpha" (ID: LOC-09)
- **Constraint:** "Maintain a 1970s analog-tech aesthetic in all reports."
- **Initial Agents:** Weaver (Tone: Paranoid/Procedural), Critic (Focus: Scientific Consistency).
