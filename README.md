# Gemini Workspace

Welcome to the Gemini Workspace. This is the root repository and development environment for a collection of projects, tools, and experiments maintained by Chris Mimms.

The directory is structured to host a variety of applications, static sites, background tasks, and AI agent experiments. It serves as a centralized hub for these distinct components.

## Github Projects

Below is a list of the core projects in this workspace, along with links to their respective GitHub repositories:

*   [**Admin Dashboard**](https://github.com/c-mimms/dashboard) (`/dashboard`): A monitoring tool to observe and administrate the various services running in the workspace.
*   [**Clicker Siege**](https://github.com/c-mimms/clicker-siege) (`/clicker-siege`): A multiplayer clicker game.
*   [**Discord Bot Brain**](https://github.com/c-mimms/discord_bot) (`/discord_bot`): Core infrastructure and Discord bot implementation for interactive services.
*   [**Personality Quiz Platform**](https://github.com/c-mimms/personality-quiz) (`/personality-quiz`): A full-stack social application for taking and sharing personality quizzes.
*   [**Timezone Map**](https://github.com/c-mimms/timezone-map) (`/timezone-map`): A static site application for visualizing and managing timezones.
*   [**Jacob's Resume**](https://github.com/c-mimms/jacob-resume) (Archived): Static site resume deployment.

## Internal Tools & Infrastructure

In addition to the separate GitHub-tracked projects, this root directory contains internal applications and tools:

*   **Background Task Scheduler** (`/background-task-scheduler`): Infrastructure for running periodic tasks and cron jobs.
*   **Job Task Agent** (`/job_agent`): General automation and agent task coordination.
*   **Agent Tools** (`/agent_tools`): Utility scripts and functions used by AI agents within the environment (e.g. email tools).
*   **Static Sites** (`/sites`): A collection of generated and maintained static websites:
    *   **Mimms Museum** (`/sites/museum`)
    *   **Georgia Mining Archive** (`/sites/georgia_mining`)
    *   **The Dispatch (News)** (`/sites/news`)
    *   **Research Archive** (`/sites/research_archive`)
    *   **Designs Portfolio** (`/sites/designs`)
